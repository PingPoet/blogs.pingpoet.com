SQL Server 2005 Setup

I wasn't able to get my upgrade to work for Analysis Services. It kept saying the upgrade was “blocked”. I had to uninstall SQL Server 2000 Analysis Services to get it to install. This was kind of a pain, but moving forward, I don't see me using AS2000 for anything really. I'm anxious to see what all is new in AS2005.

Oh yeah, be sure to turn off your virus protection and/or firewall apps (ie. McAfee) before install, otherwise you will have big problems... Don't ask me how I know.
