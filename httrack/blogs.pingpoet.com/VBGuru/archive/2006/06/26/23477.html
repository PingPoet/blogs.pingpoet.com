Hierarchical News Aggregator

I found the coolest link today: http://www.marumushi.com/apps/newsmap/index.cfm

This is a web page that reads the latest news with the Google News API and displays the results visually in a hierarchical treemap using a flash front end.

The bigger blocks indicate that there are more "related articles" about that topics and the color of the blocks tell you how recent the news is. You can filter what type of news you want by using the checkboxes at the bottom. The tabs across the top allow you to filter which country... How cool is that?

A cool feature request would be to allow the API to query the user's personalized news page (old post).
