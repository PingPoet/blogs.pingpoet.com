ActiveReports usage on the web

ActiveReports report documents have a default title of “ActiveReports Document”.  Many users do not change that default value and it ends up part of HTML and PDF exports.  I used this bit of information to see how our customers use ActiveReports on the web.   A simple google search for the string yielded 10 pages of results.  In those results, some cool reports that are available on the web as example uses of our popular product.  Cool.
